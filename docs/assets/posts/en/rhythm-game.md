## How It Works

- A **beat map** is generated by recording key press timings relative to the music.
- These timestamps determine when each visual note should reach its target.
- During gameplay, the system compares the userâ€™s inputs to the note positions in real time.
- Scoring is based on timing precision â€” ranging from **Miss** to **Perfect**.
